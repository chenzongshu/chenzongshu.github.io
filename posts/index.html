启动 启动从源码 /etcdmain/main.go 中的main函数开始 func Main() { checkSupportArch() // 检查系统是否支持 if len(os.Args) > 1 { // 获取入参 cmd := os.Args[1] // 获取启动命令 if covArgs := os.Getenv("ETCDCOV_ARGS"); len(covArgs) > 0 { args := strings.Split(os.Getenv("ETCDCOV_ARGS"), "\xe7\xcd")[1:] rootCmd.SetArgs(args) cmd = "grpc-proxy" } switch cmd { case "gateway", "grpc-proxy": if err := rootCmd.Execute(); err != nil { fmt.Fprint(os.Stderr, err) os.Exit(1) } return } } startEtcdOrProxyV2() } 上面仔细去看，前面是根据命令行输入的第一个参数去启动不同的代码逻辑 后面的startEtcdOrProxyV2()则是启动etcd server的函数 命令行输入 我们来查看一下etcd命令的使用帮助: $ ./etcd -help Usage: etcd [flags] Start an etcd server.……
